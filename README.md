## Smart Expense Analyzer & Spending Predictor
**Project Description**
This project analyzes personal expenses using different categories like food, travel, shopping, bills, and others. It demonstrates a complete data science workflow including data preprocessing, visualization, statistical analysis, and machine learning models to understand and predict spending behavior.

**Objectives**
Understand expense dataset structure
Perform data cleaning and preprocessing
Analyze spending patterns
Visualize data using graphs and heatmap
Predict future expenses
Classify high and low spenders

**Technologies Used**
Python
NumPy
Pandas
Matplotlib
Seaborn
Scikit-learn
SciPy

**Techniques Implemented**
Data Cleaning
Handling Missing Values (NaN)
Correlation Heatmap
Linear Regression
Logistic Regression
Z-test (Hypothesis Testing)
Data Sampling
Trend Analysis & Visualization
Simulation of Future Expenses

**Output**
Correlation Heatmap
Expense Trend Graph
Regression Line Graph
Confusion Matrix
Z-test Results
Future Expense Simulation

**File Included**
Smart_Expense_Analyzer.ipynb 

**Conclusion**
This project successfully analyzes and predicts personal spending patterns using data science techniques. It helps in understanding financial behavior and supports better budgeting and decision-making.
